# 🎯 Number Guessing Game

A simple Python game where the computer randomly selects a number between 1 and 100, and the player must guess it.

## Description

This project generates a random number and continuously asks the user to guess it until the correct number is entered. The program provides hints after each incorrect guess.

## Features

- Random number generation
- User input handling
- Hint system
  - Too low! Try again.
  - Too high! Try again.
- Unlimited attempts
- Beginner-friendly code

## Technologies Used

- Python 3
- Random Module

## How to Run

1. Clone the repository:

git clone https://github.com/your-username/number-guessing-game.git

2. Navigate to the project folder:

cd number-guessing-game

3. Run the program:

python main.py

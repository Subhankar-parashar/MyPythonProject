# 🧮 Python Calculator

A simple calculator application built using Python and Tkinter that performs basic mathematical operations through popup input dialogs.

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 📊 Average calculation
- 🔢 Percentage calculation
- 🧠 Factorial calculation
- 🪟 GUI popup input using Tkinter

## 🛠 Technologies Used

- Python
- Tkinter

## 🚀 How to Run

1. Clone the repository

```bash
git clone <your-repo-link>
```

2. Navigate to the project folder

```bash
cd MyPythonProject
```

3. Run the calculator

```bash
python calculator.py
```

## 📷 How It Works

- User enters numbers through popup dialogs
- User selects an operation
- Program performs calculation
- Result is displayed in a popup message box

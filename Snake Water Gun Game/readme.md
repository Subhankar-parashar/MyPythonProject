Snake Water Gun Game

Description
-----------
This is a simple Snake Water Gun game built using Python. The player competes against the computer for 5 rounds. The computer randomly selects Gun, Snake, or Water, and the winner of each round is decided according to the game rules. At the end of 5 rounds, the final winner is announced based on the scores.

Rules
-----
1. Snake drinks Water -> Snake wins
2. Water douses Gun -> Water wins
3. Gun kills Snake -> Gun wins
4. If both choices are the same, the round is a tie.

Features
--------
- Random computer choices using Python's random module
- 5 rounds of gameplay
- Score tracking for player and computer
- Final winner announcement
- Simple command-line interface

Requirements
------------
- Python 3

How to Run
----------
1. Download or clone the repository.
2. Open a terminal in the project folder.
3. Run:

python main.py

Example Gameplay
----------------
Enter your choice (Gun/Snake/Water): Gun
Computer chose: Snake
You won this round!

Enter your choice (Gun/Snake/Water): Water
Computer chose: Water
It's a tie!

Final Score
You: 3
Computer: 1
You won the game!

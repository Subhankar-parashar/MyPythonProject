# 🔐 Code Language Generator

A fun Python project that converts normal text into a scrambled "secret language" by randomly rearranging the letters of each word.

## 🚀 Features

- Takes user input from the terminal
- Splits the sentence into individual words
- Randomly shuffles the letters of each word
- Generates a unique encoded sentence every time
- Beginner-friendly Python project

## 🛠️ Technologies Used

- Python 3
- Random Module

## 📖 How It Works

1. The user enters a sentence.
2. The program splits the sentence into words.
3. Each word is converted into a list of characters.
4. The characters are shuffled randomly.
5. The shuffled letters are joined back into words.
6. The encoded sentence is displayed.

## 💻 Example

Input:

hello world

Output:

olhel rdowl

(Note: Output changes every time because the letters are shuffled randomly.)

## ▶️ Run the Program

bash python main.py

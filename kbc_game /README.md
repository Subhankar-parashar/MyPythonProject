# KBC Game 🎮

A simple Python-based Kaun Banega Crorepati (KBC) quiz game made using Python.

## Features
- Multiple-choice questions
- Prize money levels
- User input handling
- Quiz progression system

## Technologies Used
- Python

## How to Run

1. Clone the repository

```bash
git clone <your-repo-link>
```

2. Open project folder

```bash
cd your-folder-name
```

3. Run the program

```bash
python kbc.py
```

Dice Roller Simulator

Overview

This Python project simulates rolling any number of six-sided dice and displays the result using ASCII art. Each die is randomly generated and its face is rendered in the terminal.

Features

* Roll multiple dice at once
* Randomly generates values from 1 to 6
* Displays dice faces using ASCII art
* Calculates and displays the total value of all dice
* Simple command-line interface

Concepts Used

* Dictionaries
* Tuples
* Lists
* Loops
* User Input
* Random Number Generation
* ASCII Art Rendering

How It Works

1. The user enters the number of dice to roll.
2. The program generates random values between 1 and 6.
3. Each die is displayed using a predefined ASCII art template.
4. The total sum of all dice is calculated and displayed.

Example

Input:

How many dice? : 3

Output:

Dice Values:
[2, 5, 1]

(ASCII Art of each die)

Total value is 8

Possible Improvements

* Display dice side-by-side instead of vertically.
* Add roll history.
* Allow different types of dice (D4, D8, D10, D20).
* Add colored terminal output.
